# chatting-server
- This is high scalability system design of realtime chatting server.

## Architecture
- Domain Driven Design
- Event Sourcing
- Test Driven Development

## dependencies
- express
- typescript
- socket.io
- sequelize

## Install
- `npm install`

## Unit Test
- `npm run test`

## Build
- `npm run build`

## run 
- dev: `npm run dev`
- production: `npm start`

## Database
- mysql
